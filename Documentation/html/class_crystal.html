Description détaillée

Cette classe modélise les cristaux utilisés dans le jeu.

Un cristal est un objet circulaire centré en un point, et d'un certain rayon.

Un rayon lumineux passant à travers un crystal modifie sa longueur d'onde (en l'augmentant ou en la diminuant d'une certaine valeur) mais pas sa trajectoire.
